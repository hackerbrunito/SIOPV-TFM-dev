## 1. WHAT STAGE 4.1 IS

Stage 4.1 is the first of two sub-stages of Claude Code Configuration Setup for SIOPV.

**Stage 4.1 produces one deliverable:** the truth document — a comprehensive, granular technical specification of everything that must exist in `siopv/.claude/`, informed by online research and all prior SIOPV audit findings.

Stage 4.1 does NOT create any directories or files in `siopv/.claude/`. It produces a document only.

---

## 2. WHAT STAGE 4.1 IS NOT

- Stage 4.1 does NOT implement anything — that is Stage 4.2
- Stage 4.1 does NOT create or write to `siopv/.claude/` — document output only
- Stage 4.1 does NOT plan Stage 4.2 execution — it only produces the truth document that Stage 4.2 consumes
- Stage 4.1 does NOT delete anything from the meta-project

## 4. THE TRUTH DOCUMENT — REQUIRED CONTENT

The truth document must be so complete that Stage 4.2 agents can implement without any inference or improvisation. For every file in every directory of `siopv/.claude/`, it must specify:

- Exact directory placement (absolute path)
- File skeleton / structure (all sections listed)
- Contents of each section — what it must contain and why
- Cross-file dependencies and connections to other files
- Whether the file is: copied as-is from meta-project / adapted from meta-project / created from scratch
- What is obsolete/superseded in the meta-project version and must NOT be carried over (with explanation of what replaced it)

## 6. ABSOLUTE RULES FOR STAGE 4.1

- All paths are hardcoded absolute paths — no `$HOME`, no relative paths
- All agents spawned with `model: "sonnet"` — never Opus
- No agent creates directories — team-lead creates all directories
- Stage 4.1 output goes to `stage4.1-research/` only — never to `siopv/.claude/`
- Online research findings take priority over meta-project patterns where they conflict
- No improvisation by any agent — if unclear, stop and report to team-lead
- Human approves at every checkpoint — orchestrator's idle state is not approval
- Round 2 does NOT begin until Round 1 is complete and human has approved

## 8. TRANSITION TO STAGE 4.2
